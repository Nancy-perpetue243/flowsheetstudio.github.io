🛠️ Comment ajouter tes produits & gérer les livraisons

Voici exactement comment fonctionne ton business numérique de A à Z — étape par étape.

1
Crée ton fichier (Sheets, PPT, PDF...)

Crée ton produit sur Google Sheets, PowerPoint, Canva ou tout autre outil. Exporte-le en PDF ou garde-le en .xlsx / .pptx. C'est ton fichier à vendre.

Outils : Google Sheets · Canva · PowerPoint
2
Upload sur Gumroad ou LemonSqueezy

Va sur gumroad.com → "New Product" → Upload ton fichier → Fixe ton prix → Copie le lien de paiement généré. C'est ça ton lien de vente.

Plateforme recommandée : Gumroad (gratuit)
3
Remplace les boutons "Acheter" du site

Dans le code HTML de ce site, chaque bouton appelle openModal(). Remplace les liens par tes vrais liens Gumroad. Ex : window.open('https://gumroad.com/l/tonproduit')

Dans le code : fonction openModal() → lien Gumroad
4
Ajoute tes nouvelles cartes produit

Dans le HTML, copie-colle un bloc <div class="prod-card">, change le nom, l'emoji, la description et le prix. Mets data-cat="sheets", "ppt", "cartes" ou "invit".

Copier-coller dans la section #prodGrid
📦

Via Gumroad ou Stripe — carte, Apple Pay, Google Pay
